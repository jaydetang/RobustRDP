# Processed Train Data

This directory contains the processed training datasets for the RobustRDP project. The data is organized into three subsets: **pretrain data**, **SFT data**, and **DPO data**.

> **Download**: The full dataset is available on Hugging Face:  
> [https://huggingface.co/datasets/Jingcz/RobustRDP-ProcessedTrainData](https://huggingface.co/datasets/Jingcz/RobustRDP-ProcessedTrainData)

---

## Directory Structure

```
processed_train_data/
│
├── pretrain_data/                          # Pretraining data (60k images)
│   ├── pretrain_downsampled_llm_6w.json           # Merged annotation file (138 MB)
│   └── images_pretrain_resized/                   # Resized reaction diagram images
│       ├── single_line_resized/                   #   10,000 images (single-line chain reactions)
│       ├── multi_line_resized/                    #   10,000 images (multi-line chain reactions)
│       ├── branch_resized/                        #   20,000 images (branching reactions)
│       └── cycle_resized/                         #   20,000 images (cyclic reactions)
│
├── sft_data/                               # SFT data
│   ├── multi_task_sft_downsampled_llm.json        # Annotation file (475 MB)
│   ├── images_train_resized/                      #    4,240 images (original real-world diagrams)
│   └── images_train_aug_resized/                  #   63,600 images (augmented variants)
│
└── dpo_data/                               # DPO data
    └── train_downsampled_llm_dpo.json              # Annotation file (69 MB)
```

## Subset Descriptions

### 1. Pretrain Data (`pretrain_data/`)

Synthetic chemical reaction diagram data generated by **LayoutDrivenSynthesizer** (see `pretrain_data_process/` for generation scripts). Contains 60,000 resized reaction images across four layout types, along with a merged JSON annotation file using the special‑token format.

| Subdirectory         | Count  | Description                     |
|----------------------|--------|---------------------------------|
| `single_line_resized` | 10,000 | Single-line chain-style reactions |
| `multi_line_resized`  | 10,000 | Multi-line chain-style reactions  |
| `branch_resized`      | 20,000 | Branching reactions                |
| `cycle_resized`       | 20,000 | Cyclic reactions                   |

**Annotation format** (in `pretrain_downsampled_llm_6w.json`):

```json
{
    "messages": [
        {
            "content": "<image> \nEach image contains ...",
            "role": "user"
        },
        {
            "content": "<rxn><rct>x1 y1 x2 y2<mol><cnd>x1 y1 x2 y2<txt><prd>x1 y1 x2 y2<mol>...",
            "role": "assistant"
        }
    ],
    "images": [
        "processed_train_data/pretrain_data/images_pretrain_resized/single_line_resized/single_line_0.png"
    ]
}
```

### 2. SFT Data (`sft_data/`)

Supervised fine-tuning data based on **real-world** chemical reaction diagrams (e.g., from literature). Includes both original resized images and their augmented counterparts.

| Subdirectory               | Count  | Description                      |
|----------------------------|--------|----------------------------------|
| `images_train_resized`     |  4,240 | Resized real-world reaction diagrams |
| `images_train_aug_resized` | 63,600 | Augmented versions (15 augments × 4,240 originals) |

The annotation file `multi_task_sft_downsampled_llm.json` follows the same special‑token format as the pretrain data.

### 3. DPO Data (`dpo_data/`)

Direct Preference Optimization data used for alignment training. The file `train_downsampled_llm_dpo.json` contains preference pairs based on the pretrain dataset.

---

## Usage

After downloading from Hugging Face, place the extracted contents so that the directory structure matches the layout above. The dataset is ready to be used with the training scripts in the repository root.
